# BuildFlow 🏗️

[![Blender](https://img.shields.io/badge/Blender-%23F5792A.svg?style=for-the-badge&logo=blender&logoColor=white)](https://www.blender.org)
[![Procedural](https://img.shields.io/badge/Procedural-Geometry%20Nodes-blue?style=for-the-badge)]()

**BuildFlow** — це потужний процедурний інструмент для Blender, призначений для автоматичного створення таймлапс-анімацій будівництва об'єктів (Full CG procedural timelapse animation). Система дозволяє генерувати реалістичну геометрію та динаміку за лічені кліки на основі вашого власного майстер-плану.

## ✨ Ключові можливості (Features)

- 🏢 **Власні будівлі та майстер-плани:** Використовуйте свої унікальні 3D-моделі для процедурної генерації.
- 📐 **Процедурна геометрія:** Автоматичне розгортання будівельних лісів, опалубки, перекриттів та конструкцій відповідно до етапів будівництва.
- 🏗️ **Автоматична анімація техніки:** Динамічні будівельні крани, екскаватори та інший транспорт оживають без ручного виставлення ключів анімації.
- 🍃 **Процедурна симуляція вітру:** Легка та швидка інтеграція симуляції вітру для рослинності у вашій сцені.
- ⚙️ **Гнучкий контроль:** Повне керування параметрами за допомогою системи Geometry Nodes у Blender.

## 🛠️ Вимоги
- Blender 4.x (або новіша версія з підтримкою Geometry Nodes)

## 🚀 Як почати

1. Завантажте цей репозиторій або актуальний `.blend` файл із розділу **Releases**.
2. Відкрийте сцену в Blender.
3. Додайте ваші моделі у відповідні колекції (Collections) або налаштуйте параметри інструменту в панелі Geometry Nodes.
4. Натисніть `Space`, щоб запустити анімацію таймлапсу!
